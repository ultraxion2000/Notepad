# Notepad
## Что представляет данное приложение:
Приложение направлено на создание своих заметок. Также предоставлена возможность добавления фотографий. 
## Использовано:
> * Kotlin
> * SQLite
> * Coroutine
> * Picaso
> * RecyclerView
